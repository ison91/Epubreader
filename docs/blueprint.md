# **App Name**: ePub Reader

## Core Features:

- File Upload and Display: Upload and display EPUB/PDF files from local storage.
- Content Pagination: Smoothly renders and paginates the book content for optimal readability.
- Navigation: Navigate through the book using page controls.
- Customizable Styles: Adjust font size and line height to improve reading comfort.
- Table of Contents and Progress: Jump to any location in the book with chapter markers and a global slider.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), evoking a sense of calmness and sophistication for reading.
- Background color: Light gray (#EEEEEE), providing a gentle contrast that is easy on the eyes during long reading sessions.
- Accent color: Teal (#009688) to highlight interactive elements and call to actions, offering a refreshing visual cue.
- Body: 'Literata', serif, giving a classic literary feel, for the bulk of the book content. Headlines: 'Inter', sans-serif.
- Clean, distraction-free reading environment that focuses on content presentation and navigation. Minimal chrome.
- Gentle transitions and animations, such as page turns or progress updates, contribute to a smoother user experience.
